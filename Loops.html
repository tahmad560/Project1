Java Loops Explained

For Loop

The for loop is used when the number of iterations is known. It allows you to iterate over a range of values by initializing, checking a condition, and incrementing or decrementing a counter.

Syntax:

for (initialization; condition; update) {
    // Code to be executed
}

Example:

for (int i = 0; i < 5; i++) {
    System.out.println("Iteration: " + i);
}

Use Case: When you know the exact number of iterations needed, such as looping through an array or performing a task a fixed number of times.

While Loop

The while loop is used when the number of iterations is not known in advance. It executes the loop body as long as the condition is true.

Syntax:

while (condition) {
    // Code to be executed
}

Example:

int count = 0;
while (count < 5) {
    System.out.println("Count: " + count);
    count++;
}

Use Case: When the loop should continue until a certain condition is met, such as waiting for user input or processing until a certain state is achieved.

Do-While Loop

The do-while loop is similar to the while loop, but it guarantees that the loop body will be executed at least once, regardless of the condition.

Syntax:

do {
    // Code to be executed
} while (condition);

Example:

int count = 0;
do {
    System.out.println("Count: " + count);
    count++;
} while (count < 5);

Use Case: When the loop body must be executed at least once, such as displaying a menu before checking user input.

For-Each Loop

The for-each loop is used to iterate through collections or arrays. It simplifies the code and makes it more readable when dealing with elements in a collection.

Syntax:

for (type element : collection) {
    // Code to be executed
}

Example:

int[] numbers = {1, 2, 3, 4, 5};
for (int num : numbers) {
    System.out.println("Number: " + num);
}

Use Case: When you need to iterate over all elements of an array or collection without needing to manage an index explicitly.
