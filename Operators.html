Operators in Java

Introduction

Operators in Java are special symbols or keywords used to perform operations on variables and values. Java supports a wide variety of operators, each serving a unique purpose in programming.

Types of Operators

  • Arithmetic Operators: Used for mathematical operations like addition (+), subtraction (-), multiplication (*), and division (/).
  • Relational Operators: Used to compare two values, such as greater than (>), less than (<), and equals (==).
  • Logical Operators: Used to perform logical operations like AND (&&), OR (||), and NOT (!).
  • Bitwise Operators: Used to perform operations at the bit level, such as AND (&), OR (|), and XOR (^).
  • Assignment Operators: Used to assign values to variables, such as =, +=, -=, and *=.
  • Unary Operators: Used with a single operand, such as increment (++), decrement (--), and negation (-).
  • Ternary Operator: A shorthand for if-else conditions (condition ? trueValue : falseValue).

Examples

Arithmetic Operators

int a = 10, b = 5;
System.out.println("Addition: " + (a + b)); // Output: 15
System.out.println("Multiplication: " + (a * b)); // Output: 50

Relational Operators

int x = 10, y = 20;
System.out.println(x > y); // Output: false
System.out.println(x <= y); // Output: true

Logical Operators

boolean p = true, q = false;
System.out.println(p && q); // Output: false
System.out.println(p || q); // Output: true

Bitwise Operators

int m = 5, n = 3;
System.out.println(m & n); // Output: 1 (Bitwise AND)
System.out.println(m | n); // Output: 7 (Bitwise OR)

Assignment Operators

int num = 10;
num += 5; // Equivalent to num = num + 5
System.out.println(num); // Output: 15

Unary Operators

int z = 10;
System.out.println(++z); // Output: 11 (Pre-increment)
System.out.println(z--); // Output: 11 (Post-decrement)

Ternary Operator

int a = 10, b = 20;
int max = (a > b) ? a : b;
System.out.println("Maximum: " + max); // Output: 20

Practice

Try solving the following problems to understand operators better:

  1. Write a program to calculate the remainder of two numbers using the modulus operator (%).
  2. Create a program to check if a number is even or odd using the relational and logical operators.
  3. Experiment with bitwise operators to manipulate binary values.
