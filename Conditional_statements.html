If-Else and Switch Case in C

If-Else

Key Points

  • Conditions: Use relational (==, !=, >, <, >=, <=) or logical operators (&&, ||, !) to form conditions.
  • Code Blocks: Always use braces {} for better readability, even for single-line statements.
  • Indentation: Indent code inside if and else blocks for clarity.
  • Efficiency: Use else if to avoid unnecessary checks once a condition is satisfied.

Syntax

Simple if Statement:

if (condition) {
    // Code to execute if condition is true
}

if-else Statement:

if (condition) {
    // Code to execute if condition is true
} else {
    // Code to execute if condition is false
}

if-else if-else Ladder:

if (condition1) {
    // Code to execute if condition1 is true
} else if (condition2) {
    // Code to execute if condition2 is true
} else {
    // Code to execute if all conditions are false
}

Examples

1. Basic if Statement:

#include <stdio.h>

int main() {
    int num = 10;
    if (num > 5) {
        printf("The number is greater than 5.\n");
    }
    return 0;
}

Output: The number is greater than 5.

2. Using if-else:

#include <stdio.h>

int main() {
    int num = 3;
    if (num > 5) {
        printf("The number is greater than 5.\n");
    } else {
        printf("The number is not greater than 5.\n");
    }
    return 0;
}

Output: The number is not greater than 5.

3. if-else if-else Ladder:

#include <stdio.h>

int main() {
    int num = 0;
    if (num > 0) {
        printf("The number is positive.\n");
    } else if (num < 0) {
        printf("The number is negative.\n");
    } else {
        printf("The number is zero.\n");
    }
    return 0;
}

Output: The number is zero.

Switch Case

Syntax

switch(expression) {
    case value1:
        // Code to execute if expression == value1
        break;
    case value2:
        // Code to execute if expression == value2
        break;
    ...
    default:
        // Code to execute if none of the cases match
}

Examples

1. Simple Switch Case:

#include <stdio.h>

int main() {
    int choice;
    
    printf("Enter a number (1-3): ");
    scanf("%d", &choice);
    
    switch(choice) {
        case 1:
            printf("You chose option 1.\n");
            break;
        case 2:
            printf("You chose option 2.\n");
            break;
        case 3:
            printf("You chose option 3.\n");
            break;
        default:
            printf("Invalid choice. Please enter a number between 1 and 3.\n");
    }
    
    return 0;
}

Output:

  • Input: 2
    Output: You chose option 2.
  • Input: 5
    Output: Invalid choice. Please enter a number between 1 and 3.

2. Switch with Character Input:

#include <stdio.h>

int main() {
    char grade;
    
    printf("Enter your grade (A, B, C): ");
    scanf(" %c", &grade);
    
    switch(grade) {
        case 'A':
            printf("Excellent!\n");
            break;
        case 'B':
            printf("Well done.\n");
            break;
        case 'C':
            printf("Good effort.\n");
            break;
        default:
            printf("Invalid grade. Please enter A, B, or C.\n");
    }
    
    return 0;
}

Output:

  • Input: A
    Output: Excellent!
  • Input: E
    Output: Invalid grade. Please enter A, B, or C.

3. Without break:

#include <stdio.h>

int main() {
    int num;
    
    printf("Enter a number (1-3): ");
    scanf("%d", &num);
    
    switch(num) {
        case 1:
            printf("Case 1\n");
        case 2:
            printf("Case 2\n");
        case 3:
            printf("Case 3\n");
        default:
            printf("Default case\n");
    }
    
    return 0;
}

Output:

  • Input: 2
    Output:
    Case 2
    Case 3
    Default case
