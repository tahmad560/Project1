Mastering Conditional Logic: If-Else, Else-If, Nested If-Else, and Switch-Case

Explore how to control the flow of your Java program using if-else, else-if, nested if-else, and switch-case statements.

Understanding If-Else and Else-If Statements

The if-else statement allows you to execute a block of code based on a condition. It's a simple way to make decisions in your code, directing it to different paths depending on whether the condition is true or false.

Syntax:

if (condition) {
    // code block if condition is true
} else {
    // code block if condition is false
}

Else-If:

The else-if statement allows you to check multiple conditions in a single block. If the first condition is false, the program checks the next condition and so on.

if (condition1) {
    // code block for condition1
} else if (condition2) {
    // code block for condition2
} else {
    // code block if no conditions are true
}

Example:

Here's an example that categorizes a person based on their age:

int age = 20;
if (age < 13) {
    System.out.println("Child");
} else if (age < 20) {
    System.out.println("Teenager");
} else {
    System.out.println("Adult");
}

Nested If-Else Statements

Nested if-else allows you to place one if or else-if statement inside another. This is useful when you need to evaluate multiple conditions in a hierarchical manner.

Syntax:

if (condition1) {
    if (condition2) {
        // code block for condition2
    } else {
        // code block for condition2 false
    }
} else {
    // code block for condition1 false
}

Example:

In this example, we first check if a person is an adult, and then if they are eligible to vote:

int age = 20;
if (age >= 18) {
    if (age >= 21) {
        System.out.println("Adult and eligible to vote");
    } else {
        System.out.println("Adult but not eligible to vote");
    }
} else {
    System.out.println("Not an adult");
}

Switch-Case Statement

The switch-case statement is used to select one of many blocks of code to be executed. It's often used when you have multiple conditions based on a single variable, making the code cleaner and easier to read.

Syntax:

switch (expression) {
    case value1:
        // code block for value1
        break;
    case value2:
        // code block for value2
        break;
    default:
        // code block if no match
}

Example:

Here's an example of a simple menu using a switch-case statement:

int choice = 2;
switch (choice) {
    case 1:
        System.out.println("Start the program");
        break;
    case 2:
        System.out.println("Exit the program");
        break;
    default:
        System.out.println("Invalid choice");
}
