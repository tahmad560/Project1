Loops in C

What is a loop in C? Loops are used to execute a block of code repeatedly as long as a certain condition is true.

Types of Loops in C

  • for Loop
  • while Loop
  • do-while Loop

1. for Loop

Syntax:

for(initialization; condition; update) {
    // Code to execute
}

Example:

#include <stdio.h>

int main() {
    for(int i = 1; i <= 5; i++) {
        printf("i = %d\\n", i);
    }
    return 0;
}

Output:

i = 1
i = 2
i = 3
i = 4
i = 5

2. while Loop

Syntax:

while(condition) {
    // Code to execute
}

Example:

#include <stdio.h>

int main() {
    int i = 1;
    while(i <= 5) {
        printf("i = %d\\n", i);
        i++;
    }
    return 0;
}

Output:

i = 1
i = 2
i = 3
i = 4
i = 5

3. do-while Loop

Syntax:

do {
    // Code to execute
} while(condition);

Example:

#include <stdio.h>

int main() {
    int i = 1;
    do {
        printf("i = %d\\n", i);
        i++;
    } while(i <= 5);
    return 0;
}

Output:

i = 1
i = 2
i = 3
i = 4
i = 5

Nested Loops in C

What is a Nested Loop? A nested loop is a loop inside another loop.

Example:

#include <stdio.h>

int main() {
    for(int i = 1; i <= 3; i++) {
        for(int j = 1; j <= 3; j++) {
            printf("i = %d, j = %d\\n", i, j);
        }
    }
    return 0;
}

Output:

i = 1, j = 1
i = 1, j = 2
i = 1, j = 3
i = 2, j = 1
i = 2, j = 2
i = 2, j = 3
i = 3, j = 1
i = 3, j = 2
i = 3, j = 3
