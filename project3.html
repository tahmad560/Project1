Recursion Practice Problems

1. Factorial Calculation

Write a recursive function to calculate the factorial of a number n.

Hint: The factorial of n is n * factorial(n-1). Base case: factorial(0) = 1.
Example:
Input: 5
Output: 120

2. Fibonacci Sequence

Create a recursive function to find the n-th Fibonacci number.

Hint: Fibonacci numbers are calculated as fib(n) = fib(n-1) + fib(n-2). Base cases: fib(0) = 0, fib(1) = 1.
Example:
Input: 6
Output: 8

3. Sum of Digits

Write a recursive function to calculate the sum of the digits of a number.

Hint: Add the last digit to the sum of the digits of the remaining number: sum(n) = n % 10 + sum(n / 10). Base case: n == 0.
Example:
Input: 123
Output: 6

4. Reverse a String

Develop a recursive function to reverse a given string.

Hint: Reverse the rest of the string and append the first character: reverse(s) = reverse(s[1:]) + s[0]. Base case: len(s) == 0.
Example:
Input: "hello"
Output: "olleh"

5. Power Calculation

Implement a recursive function to calculate a^b (a raised to the power b).

Hint: Multiply a by the result of a^(b-1). Base case: a^0 = 1.
Example:
Input: 2, 3
Output: 8

6. Palindrome Check

Write a recursive function to check whether a given string is a palindrome.

Hint: Check if the first and last characters are the same, then check the substring in between: isPalindrome(s) = s[0] == s[-1] and isPalindrome(s[1:-1]). Base case: len(s) <= 1.
Example:
Input: "radar"
Output: true

7. Tower of Hanoi

Solve the Tower of Hanoi problem for n disks.

Hint: Move n-1 disks to the auxiliary rod, move the last disk to the target rod, and then move the n-1 disks from the auxiliary rod to the target rod.
Example:
Input: 3 disks
Output: A sequence of moves transferring all disks from source to target.

8. Count Occurrences

Create a recursive function to count the occurrences of a specific element in an array.

Hint: Check if the first element matches, then count occurrences in the rest of the array. Base case: len(array) == 0.
Example:
Input: [1, 2, 3, 2, 2], 2
Output: 3

9. Binary Search

Implement the binary search algorithm using recursion.

Hint: Compare the middle element with the target and search in the appropriate half. Base case: start > end.
Example:
Input: [1, 2, 3, 4, 5], 3
Output: 2 (index)

10. Permutations of a String

Write a recursive function to generate all permutations of a given string.

Hint: Swap each character with the first character and recursively find permutations of the rest. Base case: len(s) == 1.
Example:
Input: "abc"
Output: ["abc", "acb", "bac", "bca", "cab", "cba"]
