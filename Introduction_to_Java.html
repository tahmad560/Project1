Introduction to Java

Learn the basics of Java programming language.

What is Java?

Java is a high-level, object-oriented programming language known for its platform independence. It's designed to be portable, secure, and robust, making it a popular choice for building large-scale enterprise applications, web applications, and Android development.

History of Java

Java was developed by James Gosling and Mike Sheridan at Sun Microsystems in 1991. It was initially called "Oak" and was later renamed to Java in 1995. The goal was to create a language that could run on a variety of devices. Java's "Write Once, Run Anywhere" (WORA) principle allows code written in Java to run on any device with a Java Virtual Machine (JVM), regardless of the underlying hardware.

Why Java?

  • Platform Independence: Java is platform-independent due to the JVM, which allows the same Java code to run on any operating system without modification.
  • Object-Oriented: Java follows the principles of object-oriented programming (OOP), such as inheritance, encapsulation, and polymorphism.
  • Security: Java provides a secure environment by allowing code to run in a "sandbox" where it is prevented from accessing unauthorized system resources.
  • Robust: Java's strong memory management and exception handling capabilities make it a reliable and stable language.
  • Multithreading: Java supports multithreading, allowing developers to write programs that can perform multiple tasks simultaneously, improving performance.

Is Java a Strongly Typed Language?

Yes, Java is a strongly typed language. This means that each variable in Java must be declared with a data type (such as int, String, or boolean), and the compiler ensures that operations on these variables are type-appropriate. For example, you cannot add a string to an integer without explicitly converting one type to another. This helps to prevent many types of bugs and errors in Java programs.

2. Variables and Data Types

A variable in Java is a container that holds data that can be changed during program execution. Each variable must be declared with a specific data type, which determines the size and type of data it can hold. Below are some of the basic data types in Java and the memory they occupy:

  • int: Used to store integer values (whole numbers). It occupies 4 bytes of memory.
  • double: Used to store floating-point numbers (numbers with decimals). It occupies 8 bytes of memory.
  • float: Used to store single-precision floating-point numbers. It occupies 4 bytes of memory. You must append an f or F to the number to indicate it's a float (e.g., 19.99f).
  • boolean: Used to store true or false values. It occupies 1 byte of memory.
  • char: Used to store single characters. It occupies 2 bytes of memory.
  • String: Used to store a sequence of characters. The memory size depends on the length of the string, as each character in a string occupies 2 bytes (UTF-16 encoding).
  • byte: Used to store very small integer values. It occupies 1 byte of memory and ranges from -128 to 127.
  • short: Used to store small integer values. It occupies 2 bytes of memory and ranges from -32,768 to 32,767.

Here is an example of variables declared with different data types:

    public static void main(String[] args) {

3. Control Flow: If-Else Statements

In Java, the if-else statement is used to execute a block of code based on a condition. If the condition evaluates to true, the block of code inside the if statement is executed. If it evaluates to false, the block of code inside the else statement (if present) is executed.

The general syntax for an if-else statement is:


if (condition) {
    // Code to be executed if condition is true
} else {
    // Code to be executed if condition is false
}

Here’s an example where we categorize a person's age into different groups:


public class AgeCategory {
    public static void main(String[] args) {
        int age = 20; // Declare an integer variable to store age

        // Using if-else to check the age category
        if (age < 13) {
            System.out.println("You are a child."); // Prints if age is less than 13
        } else if (age < 18) {
            System.out.println("You are a teenager."); // Prints if age is between 13 and 17
        } else {
            System.out.println("You are an adult."); // Prints if age is 18 or more
        }
    }
}

4. Loops

Loops are used to execute a block of code repeatedly. Java provides three types of loops: for, while, and do-while.

For Loop


public class ForLoop {
    public static void main(String[] args) {
        for (int i = 0; i < 5; i++) {
            System.out.println("Iteration " + (i + 1));
        }
    }
}

While Loop


public class WhileLoop {
    public static void main(String[] args) {
        int i = 0;
        while (i < 5) {
            System.out.println("Iteration " + (i + 1));
            i++;
        }
    }
}

Do-While Loop

The do-while loop in Java guarantees that the code inside the loop is executed at least once, regardless of the condition.


public class DoWhileLoop {
    public static void main(String[] args) {
        int i = 0;
        do {
            System.out.println("Iteration " + (i + 1));
            i++;
        } while (i < 5);
    }
}

4. For-each Loop

The for-each loop in Java is a simplified version of the traditional for loop. It's used to iterate through all elements in an array or collection without needing an explicit index.

The syntax for the for-each loop is:

for (elementType element : collection) {
    // Code to be executed with each element
}

Here’s an example of using the for-each loop to iterate through an array:


public class ForEachLoopExample {
    public static void main(String[] args) {
        // Declare an array of integers
        int[] numbers = {1, 2, 3, 4, 5};

        // Using for-each loop to iterate through the array
        for (int number : numbers) {
            System.out.println(number);  // Prints each element of the array
        }

        // Declare an array of Strings
        String[] fruits = {"Apple", "Banana", "Cherry"};

        // Using for-each loop to iterate through the array of Strings
        for (String fruit : fruits) {
            System.out.println(fruit);  // Prints each fruit name
        }
    }
}

When to Use:

  • The for-each loop is best when you don’t need the index of the elements.
  • It is more readable and less error-prone compared to traditional loops.
  • Use it when working with arrays or collections, where you simply need to access each element.

5. Methods

In Java, a method is a block of code that performs a specific task. Methods are defined inside a class and are used to execute a certain operation.

Method Declaration


public class MyClass {
    public static void myMethod() {
        // Method code goes here
        System.out.println("Hello from the method!");
    }
}

Method Calling

Methods can be called from the main method or other methods. It is common to place methods below the main method.


public class MyProgram {
    public static void main(String[] args) {
        // Calling a helper method from below
        printMessage();
    }

    // Helper method placed below the main method
    public static void printMessage() {
        System.out.println("This is a helper method.");
    }
}

6. Naming Conventions

Java has standard naming conventions that improve the readability and maintainability of the code:

  • Classes: Class names should be written in CamelCase starting with an uppercase letter (e.g., MyClass).
  • Methods: Method names should be written in camelCase starting with a lowercase letter (e.g., printMessage).
  • Variables: Variable names should also be in camelCase (e.g., myAge, userName).

Taking Input in Java

In Java, we use the Scanner class to take input from the user. The Scanner class is part of the java.util package, so we need to import it at the beginning of our program.

To take input, we create an instance of the Scanner class and use its methods such as nextInt(), nextDouble(), nextBoolean(), and nextLine() to read different types of data.

Example: Taking Input of Different Types


import java.util.Scanner;

public class TakeInput {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Taking integer input
        System.out.print("Enter your age: ");
        int age = scanner.nextInt();

        // Taking double input
        System.out.print("Enter your height: ");
        double height = scanner.nextDouble();

        // Taking boolean input
        System.out.print("Are you a student? (true/false): ");
        boolean isStudent = scanner.nextBoolean();

        // Taking single character input (correct way)
        System.out.print("Enter your grade: ");
        char grade = scanner.next().charAt(0); // Using charAt(0) to take single character input

        // Taking string input (nextLine() for full string)
        scanner.nextLine();  // Consume the leftover newline
        System.out.print("Enter your name: ");
        String name = scanner.nextLine();

        // Displaying the input values
        System.out.println("Age: " + age);
        System.out.println("Height: " + height);
        System.out.println("Student: " + isStudent);
        System.out.println("Grade: " + grade);
        System.out.println("Name: " + name);
    }
}

Important: When taking input with Scanner, it's important to choose the right method for the type of data you're reading:

  • Use next() for reading a single word (no spaces): This method reads the next token (word) until a space, tab, or newline is encountered. It’s ideal for reading strings without spaces, like names or short answers.
  • Use nextLine() for reading entire lines: This method reads everything from the current position to the end of the line, including spaces. It's great for reading full sentences or multi-word input.
  • Use nextInt(), nextDouble(), or nextBoolean() for reading specific types of data: These methods allow you to read integers, floating-point numbers, and boolean values directly.
  • For reading a single character, use next().charAt(0): You cannot use next() directly to capture a single character, as it reads the entire word. To capture a single character, use charAt(0) on the result of next(), which extracts the first character of the word entered by the user.

For example:


char grade = scanner.next().charAt(0); // Correct way to take single character input

In the example above, next() reads a word (like "A+"), and then charAt(0) extracts just the first character ("A").

Why not use next() for characters? If you use next() alone, it will read the entire word, and if the user enters more than one character (e.g., "A+" instead of just "A"), next() will return the entire string. To avoid this, we extract just the first character using charAt(0).
