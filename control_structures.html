Conditional Statements in C++

1. if Statement

if (condition) { // code to execute if condition is true }

Example: Checking if a number is positive

int num = 5; if (num > 0) { cout << "Number is positive"; }

Flow: Executes code block only if condition evaluates to true

2. if-else Statement

if (condition) { // code if true } else { // code if false }

Example: Even/Odd check

int num = 4; if (num % 2 == 0) { cout << "Even"; } else { cout << "Odd"; }

Flow: Provides alternative execution path when condition is false

3. else-if Ladder

if (condition1) { // code 1 } else if (condition2) { // code 2 } else { // default code }

Example: Grade classification

int score = 85; if (score >= 90) { cout << "A"; } else if (score >= 80) { cout << "B"; } else if (score >= 70) { cout << "C"; } else { cout << "D"; }

Flow: Checks multiple conditions sequentially

4. switch Statement

switch (expression) { case value1: // code break; case value2: // code break; default: // default code }

Example: Day of week

int day = 3; switch (day) { case 1: cout << "Monday"; break; case 2: cout << "Tuesday"; break; // ... other cases default: cout << "Invalid day"; }

Note: Works with integral types (int, char, enum)

Best Practices

  • Use braces {} even for single-line blocks
  • Order conditions from most specific to most general
  • Use switch for multiple equality checks
  • Avoid deep nesting (consider early returns)

Common Mistakes

  • Using assignment (=) instead of equality (==)
  • Forgetting break in switch cases
  • Floating-point comparisons without tolerance
  • Unreachable code due to condition ordering
