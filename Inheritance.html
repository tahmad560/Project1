Inheritance in Java

What is Inheritance in Java?

Inheritance is a fundamental concept in Object-Oriented Programming (OOP) that allows one class (child class) to inherit fields and methods from another class (parent class). This helps to promote reusability of code. The child class can access the properties and behaviors (methods) of the parent class, and it can also override or add new functionalities.

In Java, a class inherits from another using the extends keyword. This allows the child class to inherit all public and protected members of the parent class, except constructors.

The super Keyword

The super keyword is used in Java to refer to the parent class. It is commonly used in two ways:

  • Calling parent class methods: You can use super to call a method from the parent class, even if it has been overridden in the child class.
  • Accessing parent class constructors: You can use super() to call a constructor from the parent class, especially when the parent class constructor takes parameters.

Java Code Example: Inheritance with super and super()

                
class Animal {
    // Instance variable
    String name;

    // Constructor
    Animal(String name) {
        this.name = name;  // Initialize the name
        System.out.println("Animal Constructor Called");
    }

    // Method
    void speak() {
        System.out.println("The animal makes a sound.");
    }
}

class Dog extends Animal {
    // Constructor of the subclass
    Dog(String name) {
        super(name);  // Calling the constructor of the parent class (Animal)
        System.out.println("Dog Constructor Called");
    }

    // Overriding the speak method
    @Override
    void speak() {
        super.speak();  // Calling the parent class method
        System.out.println("The dog barks.");
    }
}

public class Main {
    public static void main(String[] args) {
        Dog dog = new Dog("Buddy");  // Creating a Dog object
        dog.speak();  // Calling the speak method of Dog
    }
}

Expected Output:

Animal Constructor Called

Dog Constructor Called

The animal makes a sound.

The dog barks.

Explanation of the Code

Inheritance: In the code above, we have two classes: Animal and Dog. The Dog class inherits from the Animal class using the extends keyword. This means the Dog class automatically has access to all non-private fields and methods of the Animal class.

super() Constructor: The super() keyword is used in the constructor of the Dog class to call the constructor of the parent class (Animal). This allows us to initialize the name field in the Animal class before executing the Dog class constructor code. If we didn't explicitly call super(), the parent class constructor would be called automatically, but itâ€™s good practice to include it when parameters are passed.

super Keyword for Method Calls: The super.speak() method in the Dog class calls the speak() method from the parent class (Animal). This demonstrates that the child class can use super to access methods from the parent class, even if they are overridden in the child class. In this case, the parent class's speak() method prints "The animal makes a sound," and the child class adds its own behavior ("The dog barks.").

Output Breakdown: When we run the program:

  1. The Dog constructor is called, which calls the parent Animal constructor first (via super(name)), initializing the name field.
  2. The output shows that the Animal constructor is called first, followed by the Dog constructor.
  3. Then, the speak() method of Dog is invoked. The super.speak() call first executes the parent class's speak() method, followed by the overridden method in the child class.
