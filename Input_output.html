How to Take Input in C

1. Using scanf()

The scanf() function is the most commonly used function for taking input in C. It reads formatted data from the standard input (keyboard).

Syntax:

scanf("format_specifier", &variable);

Example:

#include <stdio.h>

int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age);
    printf("Your age is: %d\n", age);
    return 0;
}

2. Taking Multiple Inputs

You can use scanf() to read multiple inputs at once.

Example:

#include <stdio.h>

int main() {
    int a, b;
    printf("Enter two numbers: ");
    scanf("%d %d", &a, &b);
    printf("You entered: %d and %d\n", a, b);
    return 0;
}

3. Taking String Input

For strings, you can use %s with scanf(). Note that it stops reading input when a space is encountered.

Example:

#include <stdio.h>

int main() {
    char name[50];
    printf("Enter your name: ");
    scanf("%s", name);
    printf("Hello, %s!\n", name);
    return 0;
}

4. Using fgets() for Strings

fgets() reads a string (including spaces) until a newline character is encountered or the specified size limit is reached.

Example:

#include <stdio.h>

int main() {
    char sentence[100];
    printf("Enter a sentence: ");
    fgets(sentence, sizeof(sentence), stdin);
    printf("You entered: %s", sentence);
    return 0;
}

5. Taking Character Input

Using scanf():

#include <stdio.h>

int main() {
    char ch;
    printf("Enter a character: ");
    scanf("%c", &ch);
    printf("You entered: %c\n", ch);
    return 0;
}

Practice Problem

6. Reading Numbers

Example for Different Data Types:

#include <stdio.h>

int main() {
    int i;
    float f;
    double d;
    printf("Enter an integer, a float, and a double: ");
    scanf("%d %f %lf", &i, &f, &d);
    printf("You entered: %d, %.2f, %.2lf\n", i, f, d);
    return 0;
}
