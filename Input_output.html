Input and Output in Java

Introduction

In Java, input and output (I/O) operations are essential for interacting with the user or external systems. Java provides several methods and classes to handle input and output efficiently.

Taking Input in Java

To take input from the user, the Scanner class is commonly used. It is part of the java.util package.

Example: Taking User Input

import java.util.Scanner;

public class InputExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Enter your name: ");
        String name = scanner.nextLine();

        System.out.print("Enter your age: ");
        int age = scanner.nextInt();

        System.out.println("Hello, " + name + ". You are " + age + " years old.");

        scanner.close();
    }
}

Explanation: The Scanner class allows us to read input of various types, such as strings, integers, and doubles, from the keyboard.

Methods for Input

  • nextLine(): Reads a complete line of text input as a string. It's commonly used for reading names, sentences, or any input containing spaces.
  • nextInt(): Reads the next integer from the input. It is used when you need to read numbers.
  • nextDouble(): Reads the next double-precision floating point number from the input.
  • nextBoolean(): Reads the next boolean value from the input (true or false).

Note: Always remember to close the Scanner object with scanner.close() to free up system resources.

Printing Output in Java

To display output in Java, the System.out.print, System.out.println, and System.out.printf methods are commonly used.

Example: Printing Output

public class OutputExample {
    public static void main(String[] args) {
        System.out.print("This is printed on the same line. ");
        System.out.println("This is printed on a new line.");
        
        String name = "John";
        int age = 25;
        System.out.printf("Name: %s, Age: %d\n", name, age);
    }
}

Explanation:

  • System.out.print: Prints output on the same line.
  • System.out.println: Prints output and moves to the next line.
  • System.out.printf: Formats the output using placeholders (e.g., %s for strings, %d for integers).
