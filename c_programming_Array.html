Comprehensive Guide to Arrays in C Programming

What is an Array?

An array is a collection of elements of the same data type stored at contiguous memory locations. Arrays allow you to store multiple values in a single variable, making data manipulation easier and more organized.

Declaration and Initialization

To declare an array in C, use the following syntax:

data_type array_name[size];

Examples:

int numbers[5];        // Declares an array of 5 integers
float scores[10];      // Declares an array of 10 floats

You can also initialize an array at the time of declaration:

int numbers[5] = {1, 2, 3, 4, 5}; // Initializes all elements

If fewer elements are provided, the remaining ones are set to zero:

int values[5] = {1, 2};  // Remaining elements are set to 0

Types of Arrays

There are three main types of arrays in C:

  • One-Dimensional Arrays: A linear collection of elements.
  • Two-Dimensional Arrays: Used to create matrices or tables.
  • Multidimensional Arrays: Arrays with more than two dimensions.

Accessing and Modifying Array Elements

Array elements are accessed using their index, which starts from 0. Use the syntax array_name[index]:

numbers[0] = 10; // Assigns 10 to the first element
printf("%d", numbers[0]); // Prints the first element

Two-Dimensional Arrays

To declare and initialize a two-dimensional array:

int matrix[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};

Access elements using two indices:

printf("%d", matrix[0][1]); // Outputs 2

Iterating Over Arrays

Use loops to traverse arrays efficiently:

for (int i = 0; i < 5; i++) {
    printf("Element %d: %d\n", i, numbers[i]);
}

Common Operations on Arrays

  • Finding Maximum and Minimum: Iterate through the array to compare values.
  • Sorting: Use algorithms like Bubble Sort or Quick Sort.
  • Searching: Implement Linear Search or Binary Search.

Example Programs

Finding the Largest Element in an Array

#include <stdio.h>

int main() {
    int numbers[5] = {3, 7, 2, 9, 5};
    int max = numbers[0];

    for (int i = 1; i < 5; i++) {
        if (numbers[i] > max) {
            max = numbers[i];
        }
    }

    printf("The largest element is: %d\n", max);
    return 0;
}

Reversing an Array

#include <stdio.h>

int main() {
    int numbers[5] = {1, 2, 3, 4, 5};
    printf("Original Array: ");

    for (int i = 0; i < 5; i++) {
        printf("%d ", numbers[i]);
    }

    printf("\nReversed Array: ");
    for (int i = 4; i >= 0; i--) {
        printf("%d ", numbers[i]);
    }

    return 0;
}

Key Points to Remember

  • Array indices always start from 0.
  • Array size must be a constant value known at compile time.
  • Use sizeof to calculate the size of an array: sizeof(array) / sizeof(array[0]).
  • Out-of-bounds access leads to undefined behavior.
