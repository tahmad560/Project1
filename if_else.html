If-Else in C

In programming, you often need your code to make decisions. For example, "If it's raining, take an umbrella; otherwise, enjoy the sunshine!" In C, we use the if-else statement to make decisions in our programs.

What is an If-Else Statement?

An if checks if a condition is true. If it is, the code inside the if block runs. If the condition is false, you can use else to run some other code instead.

How Does It Look?

Here’s the structure of an if and else:

If Only

if (condition) {
    // Do this if the condition is true
}

If and Else

if (condition) {
    // Do this if the condition is true
} else {
    // Do this if the condition is false
}

Example: Checking a Number

Let’s say we want to check if a number is bigger than 10.

#include <stdio.h>

int main() {
    int number = 15;

    if (number > 10) {
        printf("The number is greater than 10.\n");
    } else {
        printf("The number is 10 or less.\n");
    }

    return 0;
}

What happens? If the number is 15 (which is greater than 10), the first message is printed. Try changing the value to test it!

If-Else If: Multiple Conditions

What if we want to check more than one condition? For example, we want to check if a number is positive, negative, or zero. We can use else if like this:

#include <stdio.h>

int main() {
    int number = 0;

    if (number > 0) {
        printf("The number is positive.\n");
    } else if (number < 0) {
        printf("The number is negative.\n");
    } else {
        printf("The number is zero.\n");
    }

    return 0;
}

This lets us handle three cases!

A Fun Example: Even or Odd?

Let’s write a program to check if a number is even or odd:

#include <stdio.h>

int main() {
    int number = 7;

    if (number % 2 == 0) {
        printf("The number is even.\n");
    } else {
        printf("The number is odd.\n");
    }

    return 0;
}

Explanation: We check if the number divided by 2 has a remainder of 0. If yes, it's even; otherwise, it's odd.

What Happens Inside an If?

  • The condition inside the if (like number > 10) must be something that results in either true or false.
  • Relational operators like >, <, == (equal to), and != (not equal to) are commonly used.

Simple Practice

Try writing a program that asks the user for their age and prints:

  • "You are a child!" if the age is less than 12.
  • "You are a teenager!" if the age is between 13 and 19.
  • "You are an adult!" for anything else.

Hint: You can use if, else if, and else to solve this!

Summary

The if-else statement is your program’s way of making decisions. It's like teaching your computer to say, "If this happens, do that. Otherwise, do something else!" Keep practicing, and you'll master it in no time!
