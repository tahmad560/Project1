Understanding Arrays in Java

What Are Arrays?

An array is a data structure that stores multiple values of the same type in a single variable. Arrays are helpful when dealing with a large number of similar values and want to manage them efficiently.

Why Use Arrays?

  • To store multiple values in a single variable.
  • Efficiently manage and manipulate data.
  • Easy to iterate using loops.

Types of Arrays in Java

  • Single-Dimensional Array: A linear array where elements are stored in a single row. 
    int[] numbers = {1, 2, 3, 4, 5};
  • Multi-Dimensional Array: Arrays with more than one dimension, often used for tables or matrices. 
    int[][] matrix = {
    {1, 2},
    {3, 4},
    {5, 6}
};

Key Operations on Arrays

  • Declaring and initializing: 
    int[] array = new int[5];
  • Accessing elements: 
    System.out.println(array[0]); // Access first element
  • Iterating using loops: 
    for (int i = 0; i < array.length; i++) {
    System.out.println(array[i]);
}

Example: Reverse an Array

int[] array = {1, 2, 3, 4, 5};
for (int i = array.length - 1; i >= 0; i--) {
    System.out.print(array[i] + " ");
}

Conclusion

Arrays are a cornerstone of Java programming, providing a way to store and manipulate collections of data. Understanding arrays will help you work with data more effectively and lay the groundwork for more advanced topics like collections and algorithms.
