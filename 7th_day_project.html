Day 7: Java Project Challenges

Welcome!

Here are the 7 Java projects you'll be tackling today. Each project includes a brief description, hints to help you get started, and a sample output for reference. Good luck and have fun coding!

Projects

1. Simple Calculator

Create a program that performs basic arithmetic operations like addition, subtraction, multiplication, and division.

Hint: Use a switch statement to handle different operations based on user input.
Sample Output: 
Enter first number: 10
Enter second number: 5
Choose operation (+, -, *, /): +
Result: 15

2. Number Guessing Game

Write a program where the user guesses a randomly generated number within a range.

Hint: Use Random class to generate a random number and while loop for attempts.
Sample Output: 
Guess a number between 1 and 10: 5
Too low! Try again.
Guess a number between 1 and 10: 7
Congratulations! You guessed the number.

3. Student Grading System

Design a program that calculates grades based on marks input by the user.

Hint: Use if-else or switch statements to assign grades.
Sample Output: 
Enter marks: 85
Grade: A

4. Multiplication Table Generator

Create a program that generates and displays the multiplication table for a given number.

Hint: Use a for loop to iterate through and calculate each result.
Sample Output: 
Enter a number: 5
5 x 1 = 5
5 x 2 = 10
...
5 x 10 = 50

5. ATM Simulator

Simulate basic ATM functions such as checking balance, depositing, and withdrawing money.

Hint: Use a do-while loop to create a menu-driven program.
Sample Output: 
1. Check Balance
2. Deposit Money
3. Withdraw Money
4. Exit
Enter your choice: 1
Your balance is $1000

6. Pattern Printing

Write a program to print patterns such as a pyramid or inverted triangle based on user input.

Hint: Use nested for loops to create the desired pattern.
Sample Output: 
*
**
***
****
*****

7. Basic BMI Calculator

Develop a program that calculates Body Mass Index (BMI) based on height and weight inputs.

Hint: Use the formula BMI = weight / (height * height) and take care of units.
Sample Output: 
Enter your weight (kg): 70
Enter your height (m): 1.75
Your BMI is 22.86
