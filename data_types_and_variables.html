Java Data Types and Variables

Introduction to Variables

Variables in Java are containers for storing data values. Each variable has a specific data type, which determines the size and layout of the variable's memory.

int age = 25;
double salary = 50000.50;
String name = "John";
Key Points:
  • Variables must be declared before use.
  • Java is a statically-typed language, meaning the data type of a variable is known at compile time.
  • Variable names are case-sensitive and cannot start with a number.

Primitive Data Types

Java has eight primitive data types, which are the building blocks for data manipulation:

Data Type Size Description Example
int 4 bytes Stores integer values 42
float 4 bytes Stores decimal values 3.14f
double 8 bytes Stores precise decimal values 3.14159
char 2 bytes Stores a single character 'A'
boolean 1 bit Stores true or false true
byte 1 byte Stores small integer values 127
short 2 bytes Stores medium integer values 32000
long 8 bytes Stores large integer values 123456789L
Note: Primitive data types are stored directly in memory and are not objects.

Non-Primitive Data Types

Non-primitive data types are more complex and include:

  • String: Represents a sequence of characters. Example: "Hello, World!"
  • Arrays: Stores multiple values of the same type. Example: int[] numbers = {1, 2, 3};
  • Classes: User-defined data types.
  • Interfaces: Blueprints for classes.
Key Points:
  • Non-primitive types are stored as references to memory locations.
  • They can be assigned null to indicate the absence of a value.
