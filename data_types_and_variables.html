Java Data Types and Variables

Introduction to Variables

Variables in Java store data with specific data types. Here's an example:

int age = 25;
double salary = 50000.50;
String name = "John";

Primitive Data Types

Java's primitive types:

Data Type Size Description Example
int 4 bytes Stores integer values 42
float 4 bytes Stores decimal values 3.14f
double 8 bytes Stores precise decimal values 3.14159
char 2 bytes Stores a single character 'A'
boolean 1 bit Stores true or false true
byte 1 byte Stores small integer values 127
short 2 bytes Stores medium integer values 32000
long 8 bytes Stores large integer values 123456789L

Non-Primitive Data Types

Non-primitive data types are more complex and include:

  • String: Represents a sequence of characters. Example: "Hello, World!"
  • Arrays: Stores multiple values of the same type. Example: int[] numbers = {1, 2, 3};
