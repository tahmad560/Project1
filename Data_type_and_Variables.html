Variables in C

What is a Variable?

  • A variable is a container for storing data values.
  • It has a name (identifier) and a type that defines what kind of data it can hold.

Declaring a Variable

To declare a variable, specify the type followed by its name:

int age;

Rules for Naming Variables

  1. You can include letters, numbers, and underscores (_).
  2. You cannot start with a number (e.g., 1name is invalid).
  3. You cannot use any keywords that are in C (e.g., int, float).
  4. C is a case-sensitive language (e.g., age and Age are different).

Examples of valid variables:


char _grade = 'A';
float salary_2023 = 55000.50;
int Age = 30;

Types of Variables

  1. Local Variable: Declared inside a function; accessible only within that function.
  2. Global Variable: Declared outside any function; accessible throughout the program.
  3. Static Variable: Retains its value between function calls.
  4. Extern Variable: Declared in another file; used in the current file with the extern keyword.

Data Types in C

What is a Data Type?

  • A data type specifies the type of data that a variable can hold.
  • C has several built-in data types.

Types of Data Types

  1. Basic Data Types:
    • Integer (int): Stores whole numbers.
    • Floating-point (float, double): Stores decimal numbers.
    • Character (char): Stores a single character.
  2. Void (void): Represents "no type" (used in functions with no return value).

Size and Range of Data Types

Data Type Size Range
char 1 byte -128 to 127
int 2 or 4 bytes -32,768 to 32,767 / -231 to 231-1
float 4 bytes ±3.4E-38 to ±3.4E+38
double 8 bytes ±1.7E-308 to ±1.7E+308

Value Conversion

int to float


int a = 5;
float b;
b = a;
printf("Value of b: %.2f\n", b); // Output: 5.00

char to int


char c = 'A'; 
int num;
num = c; 
printf("ASCII value of '%c': %d\n", c, num); // Output: ASCII value of 'A': 65

Type Conversion (Type Casting)

int to float


int a = 5, b = 2;
float result = (float)a / b; 
printf("Result: %.2f\n", result); // Output: 2.50
