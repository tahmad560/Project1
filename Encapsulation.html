Understanding Encapsulation in Java with Getters and Setters

What is Encapsulation?

Encapsulation is a fundamental concept in Object-Oriented Programming (OOP). It involves the practice of bundling the data (fields) and the methods that operate on the data into a single unit, or class. Encapsulation hides the internal state of the object and only exposes the necessary functionality through public methods, typically **getters** and **setters**.

  • Fields (instance variables) are kept private.
  • Public getter and setter methods are provided to access and modify those fields.

Java Code Example: Encapsulation with Getter and Setter Methods

                
class Person {
    // Private fields
    private String name;
    private int age;

    // Getter method for name
    public String getName() {
        return name;
    }

    // Setter method for name
    public void setName(String name) {
        this.name = name;
    }

    // Getter method for age
    public int getAge() {
        return age;
    }

    // Setter method for age
    public void setAge(int age) {
        if (age >= 0) {
            this.age = age;
        } else {
            System.out.println("Age cannot be negative.");
        }
    }
}

public class Main {
    public static void main(String[] args) {
        Person person = new Person();
        
        // Setting values using setter methods
        person.setName("John");
        person.setAge(30);

        // Accessing values using getter methods
        System.out.println("Name: " + person.getName());
        System.out.println("Age: " + person.getAge());
    }
}

Expected Output:

Name: John

Age: 30

Explanation of the Code

Encapsulation: In the code above, the fields name and age in the Person class are private. This means they cannot be directly accessed from outside the class. Instead, public getter and setter methods are provided to access and modify their values.

Getter Methods: A getter method is used to retrieve the value of a private field. For example, the getName() method returns the value of the name field.

Setter Methods: A setter method is used to update the value of a private field. In the setName(String name) method, the name field is updated with the value passed as a parameter. Similarly, the setAge(int age) method allows the age to be set, but only if it is a valid non-negative value.

Validation in Setter: In the setAge method, a validation check ensures that the age is not set to a negative value. If a negative value is passed, an error message is displayed, protecting the integrity of the age field.

Output Breakdown: When we run the program:

  1. We create an instance of the Person class.
  2. We set the name and age using the setter methods.
  3. We access the name and age values using the getter methods and print them to the console.
