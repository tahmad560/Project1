Understanding Access Modifiers in Java

What are Access Modifiers?

Access modifiers in Java define the visibility or scope of classes, methods, and variables. They control where these elements can be accessed from.

Types of Access Modifiers

Modifier Class Package Subclass World
public ✅ ✅ ✅ ✅
protected ✅ ✅ ✅ ❌
default ✅ ✅ ❌ ❌
private ✅ ❌ ❌ ❌

Examples

1. Public Modifier

public class PublicExample {
    public void display() {
        System.out.println("This is a public method!");
    }
}

2. Private Modifier

public class PrivateExample {
    private String message = "This is private!";

    private void displayMessage() {
        System.out.println(message);
    }
}

3. Protected Modifier

package mypackage;

public class ProtectedExample {
    protected void display() {
        System.out.println("This is a protected method!");
    }
}

4. Default Modifier

class DefaultExample {
    void display() {
        System.out.println("This is a default access method!");
    }
}
